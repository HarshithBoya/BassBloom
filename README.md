# 🎵 BassBloom

**BassBloom** is a fully responsive music band / concert website with smooth UI, Google OAuth login, and dynamic sections like Services, About, Contact, and more.

## 🔧 Built With
- HTML, CSS, JavaScript
- Google OAuth 2.0 Login
- Node.js (for backend if used)
- Netlify (for hosting)
